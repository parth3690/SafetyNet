# Circle App - Setup Instructions

## Quick Start Guide

This is a React Native app that runs on both iOS and Android. Follow these steps to get it running.

## Prerequisites

### Required Software

1. **Node.js** (v18 or higher)
   - Download from [nodejs.org](https://nodejs.org/)

2. **For iOS Development:**
   - macOS with Xcode (latest version)
   - Xcode Command Line Tools: `xcode-select --install`
   - CocoaPods: `sudo gem install cocoapods`

3. **For Android Development:**
   - Android Studio
   - Java Development Kit (JDK 11 or higher)
   - Android SDK (API level 33+)
   - Set up Android environment variables:
     ```bash
     export ANDROID_HOME=$HOME/Library/Android/sdk
     export PATH=$PATH:$ANDROID_HOME/emulator
     export PATH=$PATH:$ANDROID_HOME/tools
     export PATH=$PATH:$ANDROID_HOME/tools/bin
     export PATH=$PATH:$ANDROID_HOME/platform-tools
     ```

## Installation Steps

### 1. Install Dependencies

```bash
# Install Node modules
npm install
# or
yarn install
```

### 2. Install iOS Dependencies (macOS only)

```bash
cd ios
pod install
cd ..
```

### 3. Install React Native CLI (if needed)

```bash
npm install -g react-native-cli
```

## Running the App

### iOS Simulator

1. Make sure you have Xcode installed
2. Open iOS Simulator from Xcode or run:
   ```bash
   open -a Simulator
   ```
3. Run the app:
   ```bash
   npm run ios
   # or
   yarn ios
   ```

### Android Emulator

1. Open Android Studio
2. Start an emulator (or connect a physical device)
3. Run the app:
   ```bash
   npm run android
   # or
   yarn android
   ```

### Development Server

In a separate terminal, start Metro bundler:

```bash
npm start
# or
yarn start
```

Press `i` for iOS or `a` for Android.

## Creating Native Projects

If you're starting from scratch, you'll need to initialize the native projects:

### Option 1: Use React Native CLI

```bash
# Initialize React Native project
npx react-native init Circle --version 0.73.0

# Then copy the src/ folder and other config files from this project
```

### Option 2: Use Expo (Easier for beginners)

```bash
# Install Expo CLI
npm install -g expo-cli

# Create new Expo project
expo init Circle

# Choose "blank" template
# Then copy src/ folder and config files
```

## Common Issues & Solutions

### Issue: "Command not found: react-native"

**Solution:**
```bash
npm install -g react-native-cli
```

### Issue: iOS build fails with CocoaPods errors

**Solution:**
```bash
cd ios
rm -rf Pods Podfile.lock
pod install
cd ..
```

### Issue: Android build fails

**Solution:**
```bash
cd android
./gradlew clean
cd ..
npm start -- --reset-cache
```

### Issue: Metro bundler cache issues

**Solution:**
```bash
npm start -- --reset-cache
```

### Issue: "Unable to resolve module"

**Solution:**
```bash
rm -rf node_modules
npm install
# For iOS:
cd ios && pod install && cd ..
```

## Building for Production

### iOS

1. Open `ios/Circle.xcworkspace` in Xcode
2. Select "Any iOS Device" as target
3. Product → Archive
4. Follow App Store Connect submission

### Android

```bash
cd android
./gradlew assembleRelease
# APK will be in android/app/build/outputs/apk/release/
```
